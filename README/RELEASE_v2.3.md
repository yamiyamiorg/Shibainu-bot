# やみちゃんBot v2.3 - リリースノート


### 概要

Discordサーバーの初心者をサポートする2つの新機能を追加しました。

## 追加機能

### 1. 初心者歓迎AIリプライ

**トリガー:** チャンネル `1464999838130245742` で「はじめまして」「よろしく」を含むメッセージ

**動作:**
- Gemini 2.0 Flash で歓迎メッセージ生成
- リプライで自動送信
- **1人1回まで**（SQLiteで管理）
- テストユーザー（`1107669393049128961`）は無制限

**例:**
```
ユーザー: はじめまして！よろしくお願いします！
Bot: ようこそ、〇〇さん！🎉
     このサーバーへの参加、ありがとうございます。
     楽しい時間を過ごしてくださいね。
     困ったことがあればいつでも質問してください！
```

### 2. 初心者VC参加通知

**トリガー:** アカウント作成から14日未満のユーザーが以下のVCに参加

**対象VC（本番サーバー）:**
- `1452111129332416512`
- `1461288337687183411`
- `1467877616844410901`

**通知先（テストサーバー）:**
- チャンネルID: `1466983702667067475`

**メンション:**
- 案内部ロール: `1452478070652141729`

**通知例:**
```
@案内部 🎤
新しいメンバー @ユーザー名 さんが **雑談VC** に参加しました！
（アカウント作成から 5 日）
```

## トラブルシューティング

### Q: 歓迎メッセージが送信されない

**確認項目:**
1. チャンネルID: `1464999838130245742`
2. キーワード: 「はじめまして」または「よろしく」
4. API Key: `echo $GEMINI_API_KEY`

### Q: VC通知が届かない

**確認項目:**
1. 対象VCかどうか確認
2. アカウント年齢が14日未満か確認
3. 通知先チャンネル `1466983702667067475` が存在するか
4. Intents が有効か（Developer Portal）

### Q: テストサーバーに通知が来ない

**原因:** 案内部ロール（`1452478070652141729`）がテストサーバーに存在しない

**対処:** メンションは失敗しますが、通知メッセージ自体は送信されます。
